# EcoFootprint Calculator

## About

This is a small Java project I made to get an idea of how our daily habits affect the environment.

The program calculates a person’s yearly carbon footprint using simple inputs like travel distance, electricity usage, and diet. It’s not meant to be perfectly accurate, but it gives a rough idea and makes you think about your lifestyle.

---

## Why I Made This

We hear a lot about climate change, but we don’t usually think about how our own habits contribute to it.

So I tried to build something simple that shows:
how normal things like travelling or eating habits can impact the environment.

---

## How It Works

The program asks the user for:

* distance travelled by car and bus/train
* electricity usage
* diet type

After that, it:

* calculates emissions
* shows total carbon footprint
* gives a breakdown
* suggests one small change to improve

---

## Features

* Simple and easy to use
